## ğŸ“¦ Installation & Setup

Follow these steps to run the project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/GkGuddu/My_Portfolio.git
    cd My_Portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    The app will typically act accessible at `http://localhost:5173/My_Portfolio/`.
